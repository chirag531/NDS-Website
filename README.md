# NDS — Narayan Digital Studio

> **Capturing Memories, Creating Stories**

A fully responsive, single-page website for **Narayan Digital Studio** — a professional photography studio established in 1965, based in Channapatna, South Bengaluru, Karnataka.

---

## 🌟 About

Narayan Digital Studio specializes in turning life's precious moments into lasting memories. With 29+ years of experience, 4800+ sessions completed, and a 98% client satisfaction rate, NDS is a trusted name in professional photography across the region.

---

## 📄 Pages & Features

| Page | Description |
|------|-------------|
| **Home** | Hero section, About, Featured Gallery, Services, Testimonials, Contact Form |
| **Collections** | Full filterable photo gallery with search and category filters |
| **Login / Register** | Client sign-in and account creation |
| **Payment** | Package selection with GST calculation and secure checkout UI |
| **Dashboard** | Client portal with bookings, album downloads, and payment history |

### ✨ Highlights

- **Animated loader** with NDS branding
- **Sticky navigation** with scroll effects and mobile hamburger menu
- **Hero section** with parallax zoom animation and live stats
- **Gallery** with filter by category (Wedding, Pre-Wedding, Birthday, Baby, Family, Maternity) and live search
- **Services** with pricing cards for 6 photography packages
- **Testimonials** from real clients
- **Contact form** with service selection and date picker
- **Payment flow** with package selection, GST calculation, UPI / Card / Net Banking / QR options, and success state
- **WhatsApp floating button** for quick reach
- **Toast notifications** for user actions
- **Fade-in scroll animations**

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure and markup |
| CSS3 | Styling, animations, responsive layout |
| Vanilla JavaScript | Page switching, gallery filtering, form handling |
| Google Fonts | Cormorant Garamond, Montserrat, Playfair Display |
| Font Awesome 6 | Icons throughout the UI |

> No frameworks. No build tools. Pure HTML/CSS/JS — open `nds.html` in any browser and it works.
